## 本地启动

```bash
yarn install
yarn dev
```

打开 [http://localhost:3000](http://localhost:3000)

## 开源贡献

Outrage网站的内容目前都是用Markdown方式精简化实现。开源贡献者只需要修改md文件夹里的内容即可。

如果需要创建一个新的md文件夹，建议使用已有script

```bash
yarn createMD
```

欢迎有文案撰写技能，对社会公正问题感兴趣的人加入我们，添加更多丰富内容

本项目完全开源，欢迎任何人用自己的服务器建立相同网站
